기본 예제

목록

각각의 태그에 대해서는 다음 장에서 자세히 배울 것입니다.

1. html 제목

제목 태그는 <h6>에서 <h1>으로 정의된다.
  1. <h1>This is a heading</h1>
  2. <h2>This is a heading</h2>
  3. <h3>This is a heading</h3>

2. html 문단(단락)

문단태그는 <p>태그로 정의 된다.
  1. <p>This is a paragraph.</p>
  2. <p>This is another paragraph.</p>

3. html 링크

링크는 <a> 태그로 정의 된다.

  1. <a href="http://www.w3schools.com">This is a link</a>
(링크 주소는 href속성에 설정하면 된다. 자세한 내용은 이후 장에서 다룬다)


4. 이미지

이미지는 <img> 태그로 정의 된다.
  1. <img src="w3schools.jpg" title="W3Schools.com" width="104" height="142">
(속성을 통하여 간단한 설명(풍선도움말)이나 크기를 지정할 수 있다)
